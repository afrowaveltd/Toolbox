# Linux and Bash

WhenItFails Setter runs on .NET 10 and can be used from Linux shells such as:

- Bash,
- Zsh,
- Fish with minor syntax changes,
- GitHub Actions Linux runners,
- GitLab CI runners,
- local terminal emulators,
- Visual Studio Code integrated terminal,
- WSL when used as a Linux environment.

This guide uses Bash.

## Requirements
